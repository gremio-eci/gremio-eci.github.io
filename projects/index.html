Projetos

Wiki

A Wiki de ECI é um espaço criado e gerenciado pelos alunos, destinado a guardar informações importantes sobre o curso. A ideia é que seja o lugar onde você possa encontrar o caminho das pedras e depois contar como foi que você superou as dificuldades que surgiram.

O que tem pra fazer

  1. Incluir material sobre as matérias
  2. Escrever tutoriais, ex: "Como usar o git"
  3. Adicionar sugestões de Projetos de Extensão maneiros
  4. Informar sobre os serviços da UFRJ: Biblioteca, Transporte, etc.
  5. Se expressar como quiser

Como participar

Muito simples: Basta visitar o repositório do projeto [link], e abrir um Pull Request com a sua contribuição.

Mais simples: Fala com alguém que tá no projeto e a gente te dá o acesso!

Fluxograma

Inspirados pelo projeto de Fluxograma Interativo do Cadu, estamos trabalhando para melhorar a ferramenta e torná-la disponível para mais cursos.

O que tem pra fazer

  1. Permitir que qualquer curso gere sua grade
  2. Fazer um crawler pra ler as grades do SIGA
  3. Melhorar a Interface

Como participar

Muito simples: Basta visitar o repositório do projeto [link], e abrir um Pull Request com a sua contribuição.

Mais simples: Fala com alguém que tá no projeto e a gente te dá o acesso!

Apoio Pedagógico

Muitos alunos ingressam no curso e passam por um choque devido a algumas carências no conteúdo do Ensino Médio. Pensando nisso, esse projeto prevê alguns momentos para que os alunos tirem dúvidas de Matemática, Física, e Programação.

O intuito desse projeto não é servir de monitoria para as disciplinas, mas ajudar com os conceitos básicos que são requisitos para cursar as matérias.
Lembrando também que existe a monitoria de apoio pedagógico para Cálculo, oferecida pelo IM-UFRJ.
